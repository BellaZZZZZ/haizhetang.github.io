# 原生JS vs. React
- 原生JavaScript操作DOM繁琐、效率低（**DOM-API操作UI**）
- 使用JavaScript直接操作DOM，浏览器会进行大量的**重绘重排**
- 原生JavaScript没有**组件化**编码方案，代码复用率低
# React特点
- 采用**组件化**模式、**声明式编码**，提高开发效率及组件复用率
- 在React Native中可以使用React语法进行**移动端开发**
- 使用**虚拟DOM**+优秀的**Diffing算法**，尽量减少与真实DOM的交互
